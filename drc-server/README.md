# DRC Server

Server DRC (Disaster Recovery Center) digunakan sebagai node cadangan yang dimonitor oleh Prometheus menggunakan Node Exporter.

Server ini berada pada subnet berbeda dan terhubung melalui Router VM.

## Informasi Server

- Hostname: drc-server
- IP Address: 10.20.2.10
- Gateway: 10.20.2.1
- OS: Ubuntu Server
- Role: Disaster Recovery Node

## Services

- Node Exporter
